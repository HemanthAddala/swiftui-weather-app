# SwiftUI Weather App

A production-ready iOS Weather App built using **SwiftUI**, **MVVM architecture**, and **REST API integration**.

## Features

* Live weather data
* City-based weather search
* JSON parsing
* Error handling
* Loading state
* Clean SwiftUI UI
* MVVM architecture

## Tech Stack

* Swift
* SwiftUI
* MVVM
* URLSession
* REST API
* JSONDecoder

## Project Structure

* ContentView.swift
* WeatherViewModel.swift
* WeatherService.swift
* WeatherResponse.swift
* WeatherAppApp.swift
